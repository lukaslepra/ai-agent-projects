# AI Agent Engineering

> 系统构建企业级 AI Agent 应用 —— 从 LLM API 调用到多 Agent 协作系统

## 关于这个仓库

这里记录我从零构建 AI Agent 工程能力的完整过程。目标不是学会用工具，而是能独立设计、开发、部署和评估生产级 Agent 系统。

**技术栈：** Python · LangGraph · LangChain · OpenAI API · RAG · ChromaDB · Docker · MCP
